# Available Blocks

This file gives editors and developers a quick overview of the main blocks available in WP BBuilder.

## Layout

- **Row** — Bootstrap row wrapper for structured layouts.
- **Column** — Responsive Bootstrap column wrapper.
- **Section** — General section wrapper for grouped content.
- **Hero** — Hero/banner style section.
- **Bootstrap Div** — Flexible wrapper with custom classes.
- **Swiper** — Slider/carousel wrapper.
- **Tabs** — Tab container.
- **Tab Item** — Individual tab content item.
- **Accordion** — Expand/collapse container.
- **Accordion Item** — Individual accordion entry.

## Content

- **Button** — Call-to-action button.
- **Card** — Single card block.
- **Cards** — Repeating card layout.
- **CTA Card** — Promotional card.
- **CTA Section** — Promotional content section.
- **Alert** — Status or notice message.
- **Badge** — Small label or marker.
- **Breadcrumb** — Hierarchy / path navigation.
- **List Group** — Grouped list content.
- **Navbar** — Navigation bar layout.
- **Progress** — Progress bar.
- **Spinner** — Loading indicator.
- **Feature List** — Feature bullets with stronger presentation.
- **Timeline** — Ordered timeline content.
- **Code Display** — Styled code output block.
- **Countdown Timer** — Time-based countdown display.
- **Fun Fact** — Number/stat style highlight.
- **Table** — Styled content table.
- **Video** — Embedded or managed video content.
- **Custom Embed** — Flexible third-party embed wrapper.
- **Inline SVG** — Inline SVG output block.
- **File** — File or document link/output block.
- **Gallery / ACF Gallery** — Visual image/video gallery support.

## Social

- **Social Follow** — Profile links for social platforms.
- **Social Share** — Share buttons.
- **Social Feeds** — Multi-platform social feed section with shortcode, embed URL, raw HTML, link, and ACF fallback options.

## Forms, filtering, and marketing

- **Dynamic Form** — Configurable form builder.
- **Mailchimp** — Newsletter signup block.
- **Login / Register** — Authentication UI block.
- **Ajax Search** — Live search interface.
- **Load More** — Incremental content loading.
- **Blog Filter** — Filtered content listing.
- **Catalogue** — Query-based content grid/list.
- **Events** — Event-focused listing block.
- **Testimonials** — Testimonial slider or display block.
- **Pricing Cards** — Pricing table / plan cards.
- **Contact Links** — Email and phone links.
- **Booking Calendar** — Booking/calendar related interface.

## Data and utility

- **Chart** — Chart-based data display.
- **Google Map** — Location/map embed block.
- **Weather** — Weather information output.
- **Varda Dienas / Name Days** — Latvian name day output.
- **Sitemap** — Sitemap style navigation.
- **Menu Option** — Structured menu/service item.
- **AI Content** — Starter AI-related content/integration block.

## ACF-oriented blocks

- **ACF Hero**
- **ACF Gallery**
- **ACF Card**

## Notes

Not every project uses every block. WP BBuilder is designed so blocks can be enabled, disabled, extended, or styled per project.
